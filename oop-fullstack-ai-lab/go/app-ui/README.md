# Go Grow Glow UI

This is the React + TypeScript frontend for the Go Grow Glow MVP.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the UI:
   ```bash
   npm run dev
   ```
3. The app will proxy `/api` to the backend at `http://localhost:8080`.

## Build

```bash
npm run build
```
